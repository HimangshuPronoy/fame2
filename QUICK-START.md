# Fame. Quick Start (5 Minutes)

## 🚀 Get Running Fast

### 1. Install & Start (1 min)
```bash
npm install
npm run dev
```

### 2. Update Database (2 min)

Go to **Supabase SQL Editor** and run these 3 files in order:

1. **`supabase/update-policies.sql`** - Enables user listing creation
2. **`supabase/clear-mock-data.sql`** - Removes demo data  
3. **`supabase/make-admin.sql`** - Makes you an admin (follow instructions in file)

### 3. Test It (2 min)

1. Visit `http://localhost:3000`
2. Click "Join" and create an account
3. Go to Dashboard → Click "Create Listing"
4. Fill out the form and submit
5. See your listing on the homepage!

## ✅ Done!

Your app is now fully functional with:
- ✅ No mock data
- ✅ Users can create listings
- ✅ You're an admin
- ✅ Search & filters working
- ✅ Smooth animations
