# Product Overview

Fame. is a premium lifestyle discovery platform for curated experiences in fitness, dining, wellness, beauty, and nightlife.

## Core Features
- Browse and search curated listings across multiple lifestyle categories
- User authentication with role-based access (user/admin)
- Save favorite listings for later
- Admin dashboard for managing listings
- Concierge service for personalized recommendations
- Journal for tracking experiences
- Partner portal for business listings

## User Roles
- **User**: Browse listings, save favorites, access dashboard
- **Admin**: Full listing management capabilities (create, update, delete)

## Data Model
- **Profiles**: User accounts with role-based permissions
- **Listings**: Business/venue entries with ratings, reviews, location, pricing
- **Saved Listings**: User bookmarks for favorite venues
